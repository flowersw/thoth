thoth.package_extract.koji module¶

Functionality copied from Koji.

Even though copying code is not nice at all, this is better to have copied. Koji requires a bunch of dependencies to be present including things like gcc. We actually do not need these just for these two functions. Assuming naming is mature and stable enough, it should be safe to have them there.
